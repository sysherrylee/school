# README

# 📅 Academic Calendar Pro | 校務行事曆生成系統

[Project Type](https://img.shields.io/badge/Project-EdTech-blue)

[Tech Stack](https://img.shields.io/badge/Stack-React_%7C_Tailwind_%7C_SheetJS-green)

## 💡 目的

**只需要輸入一次資料， 就能自動生成多種符合教育現場需求的專業格式，減少排版出錯問題。**

## 🚀 特色/核心功能 (Key Features)

- 會依照設定的學期起訖日期抓對應日曆，並**自動計算**學生每月的法定出席天數。
- 多種匯出格式：支援三種符合校務需求的列印版型
* **行政報表版**：按處室分欄，適合校務會議討論。
* **教育局指定版**：連續週表格，符合官方公文陳報格式。
* **視覺化月曆版**：適合張貼於教室公佈欄。
- 支援 Google日曆同步：一鍵匯出 `.ics` 檔案
- **文書自動化**：支援匯出 Word 版行事清單與 Excel 資料庫管理。
- **備份**

## 🛠️ 技術細節 (Technical Highlights)

- **前端框架**：React 18 (Functional Components, Hooks)。
- **樣式處理**：Tailwind CSS 實現高度自定義的 A4 預覽版型與列印優化。
- **外部整合**：
    - **SheetJS**: 實現 Excel 資料的雙向導入與導出。
    - **Lucide-react**: 提供一致性的 UI 視覺符號。
